How Provably Fair Works

Matrix Spins Casino uses an advanced cryptographic system to ensure that every spin is fair, transparent, and verifiable. Our provably fair mechanism uses three components that work together to generate unpredictable and unmanipulatable game outcomes:

The Three Components

1

Server Seed (Casino-Controlled)

Before each spin, the casino generates a unique server seed and creates a SHA-256 hash of it. This hash is revealed to you before the spin, ensuring the casino cannot change it after seeing the outcome. The actual server seed is kept secret until after the spin.

2

Client Seed (Player-Controlled)

You can optionally provide a client seed—any random string of characters. This ensures you have control over one part of the outcome generation. If you don't provide one, the casino generates a random client seed for you.

3

Nonce (Spin Counter)

A nonce is simply a counter that increments with each spin on your account. Combined with the server seed and client seed, it ensures each spin produces a unique result. The same seeds with different nonces produce entirely different outcomes.

The Verification Process

  1. Before your spin: The casino displays the SHA-256 hash of the server seed. This commitment cannot be changed.
  2. You spin: Your client seed, the server seed hash, and the nonce are combined and hashed using SHA-256 to generate a result.
  3. After your spin: The casino reveals the actual server seed. You can verify that its SHA-256 hash matches the pre-spin hash.
  4. You verify: Using the verification tool below, input the server seed, client seed, and nonce to compute the exact same result. If it matches, the spin is proven fair.

Why This Matters

This system is mathematically impossible to manipulate. Even if the casino wanted to cheat, they cannot change the server seed after revealing its hash. Likewise, players cannot predict or influence outcomes without knowing the actual server seed before the spin. Every outcome is determined by cryptographic mathematics, not arbitrary casino decisions.

Understanding RTP

Important: RTP is calculated over millions of spins and represents a theoretical average. Individual sessions will vary significantly. RTP does not guarantee any specific outcome for any single player or session. It is a long-term statistical measure used to evaluate game fairness.

How RTP is Calculated

  • RTP is determined by the game's mathematical design and payout structure
  • It is calculated by dividing total payouts by total wagered amounts over a very large sample size
  • All games are tested by independent third parties to verify RTP accuracy
  • RTP values are updated regularly based on actual game performance data
